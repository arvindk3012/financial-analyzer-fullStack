ğŸ”§ Troubleshooting

Common Issues:
  • Port conflicts: Backend uses 8000, Frontend uses 3001
  • Module not found: Ensure all dependencies are installed
  • CORS errors: Make sure both servers are running
  • Upload issues: Check file format (CSV/Excel only)
  • Git push rejected: Pull latest changes first with git pull origin main
  • Merge conflicts: Resolve conflicts manually, then commit

Initial Setup (if not done):

# Initialize git repository (if needed) git init # Add remote repository git remote add origin <your-repository-url> # Configure user (if not done globally) git config user.name "Your Name" git config user.email "your.email@example.com"

Daily Git Workflow:

  1. Check current status:
    git status
  2. Pull latest changes:
    git pull origin main
  3. Add your changes:
    # Add specific files git add filename.js # Or add all changes git add .
  4. Commit your changes:
    git commit -m "Description of your changes"
  5. Push to repository:
    git push origin main
