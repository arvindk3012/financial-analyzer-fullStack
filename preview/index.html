üöÄ Quick Start Guide (Windows)

‚ö†Ô∏è Current Status: Application runs in development mode outside Docker due to container startup issues.

Prerequisites:

  • Python 3.8+ installed
  • Node.js 16+ and npm installed
  • Git for version control

Step 1: Backend Setup

  1. Navigate to backend directory:
    cd "c:\Users\Arvind Kumar\CascadeProjects\financial-analyzer-fullstack\backend"
  2. Install Python dependencies:
    pip install fastapi uvicorn python-multipart sqlalchemy asyncpg pydantic openai redis celery pandas numpy python-dotenv httpx psycopg2-binary
  3. Start the backend server:
    python simple_main.py
    ‚úÖ Backend will run on: http://127.0.0.1:8000

Step 2: Frontend Setup

  1. Open NEW terminal and navigate to frontend:
    cd "c:\Users\Arvind Kumar\CascadeProjects\financial-analyzer-fullstack\frontend"
  2. Install Node.js dependencies:
    npm install
  3. Start the React development server:
    npm start
    ‚úÖ Frontend will run on: http://localhost:3001

Step 3: Access the Application

  • Frontend UI: http://localhost:3001
  • Backend API: http://127.0.0.1:8000
  • API Documentation: http://127.0.0.1:8000/docs

üîß Troubleshooting

Common Issues:
  • Port conflicts: Backend uses 8000, Frontend uses 3001
  • Module not found: Ensure all dependencies are installed
  • CORS errors: Make sure both servers are running
  • Upload issues: Check file format (CSV/Excel only)
  • Git push rejected: Pull latest changes first with git pull origin main
  • Merge conflicts: Resolve conflicts manually, then commit

üìö Git Version Control

Initial Setup (if not done):

# Initialize git repository (if needed) git init # Add remote repository git remote add origin <your-repository-url> # Configure user (if not done globally) git config user.name "Your Name" git config user.email "your.email@example.com"

Daily Git Workflow:

  1. Check current status:
    git status
  2. Pull latest changes:
    git pull origin main
  3. Add your changes:
    # Add specific files git add filename.js # Or add all changes git add .
  4. Commit your changes:
    git commit -m "Description of your changes"
  5. Push to repository:
    git push origin main

Useful Git Commands:

# View commit history git log --oneline # Create and switch to new branch git checkout -b feature-branch-name # Switch between branches git checkout main git checkout feature-branch-name # Merge branch to main git checkout main git merge feature-branch-name # Discard local changes git checkout -- filename.js # View differences git diff
